# warehouse_management
# Система управления складом

Полноценная веб-система для управления складскими запасами с возможностью учета товаров, контроля остатков, создания накладных и генерации PDF-документов.

##  Возможности

### Для всех пользователей:
-  Регистрация и авторизация
-  Просмотр списка товаров с фильтрацией
-  Поиск товаров по названию и артикулу
-  Детальная информация о каждом товаре
-  Создание накладных на списание товаров
-  Просмотр истории накладных
-  Скачивание PDF-версий накладных

### Для администраторов:
-  Полный CRUD для товаров (создание, чтение, обновление, удаление)
-  Управление категориями товаров
-  Оформление прихода товаров на склад
-  Оформление расхода товаров
-  Просмотр всех движений товаров
-  Загрузка изображений товаров
-  Установка минимального остатка для товаров

### Особенности:
-  Умный поиск с автодополнением
-  Визуальная индикация остатков (норма/ниже минимума/отсутствует)
-  Адаптивный дизайн для мобильных устройств
-  Генерация профессиональных PDF-накладных
-  Подсчет общей стоимости товаров на складе
-  Система уведомлений о действиях

##  Технологии

### Backend:
- **Python 3.11+**
- **Django 4.2** - основной фреймворк
- **Django ORM** - работа с базой данных
- **SQLite** - база данных (легко заменяется на PostgreSQL)
- **ReportLab** - генерация PDF-документов
- **Pillow** - работа с изображениями

### Frontend:
- **Bootstrap 5** - адаптивный интерфейс
- **jQuery** - AJAX запросы
- **Bootstrap Icons** - иконки
- **CSS3** - кастомные стили

### Дополнительно:
- **django-crispy-forms** - красивые формы
- **crispy-bootstrap5** - интеграция с Bootstrap 5

##  Установка

### Предварительные требования
- Python 3.11 или выше
- pip (менеджер пакетов Python)
- Git (опционально)

Пошаговая инструкция
1. Клонировать репозиторий
bash
git clone https://github.com/yourusername/warehouse-management.git
cd warehouse-management
2. Создать виртуальное окружение
Для Windows:

bash
python -m venv venv
venv\Scripts\activate
Для Linux/Mac:

bash
python3 -m venv venv
source venv/bin/activate
После активации в терминале появится (venv) - это значит, что виртуальное окружение работает.

3. Установить зависимости
bash
pip install -r requirements.txt
4. Применить миграции
bash
python manage.py makemigrations
python manage.py migrate
5. Создать суперпользователя (администратора)
bash
python manage.py createsuperuser
Система запросит:

Username: - введите имя пользователя (например, admin)

Email address: - введите email (например, admin@example.com)

Password: - введите пароль (символы не отображаются)

Password (again): - повторите пароль

6. Собрать статические файлы
bash
python manage.py collectstatic
Когда появится запрос Type 'yes' to continue, or 'no' to cancel:, введите yes

7. Запустить сервер разработки
bash
python manage.py runserver
8. Открыть в браузере
Перейдите по адресу:

text
http://127.0.0.1:8000
Быстр
